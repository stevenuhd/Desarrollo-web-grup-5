Nuestro  proyecto  tiene como objetivo entregar una  solucion  a  la problematica de cada  empresa  en  particular
en este  caso  es  poder  organizar bajo  un solo  sistema los servicios   entregados  a  los  clientes,    en  el 
ambito de canalizar la informacion, planificar oportunamente actividades con clientes, altas y bajas de clientes y
usuarios identificacion de nuestros profesionales, reportes, revision de documentos.
Proporcionamos un sistema de informacion amigable y realmente adaptado a sus necesidades.

Contenido:
Pagina index.html 			Presentacion de nuestro sistema.
Pagina contacto.html 			Formulario de contacto para realizar consultas y solicitudes.
Pagina crearcapacitacion.html		Formulario para agendar una capacitación en el sistema. 
Pagina Listarcapacitaciones 		Listado de capacitaciones registradas.
Pagina listadousuarios.html 		Crea usuario segun perfil de Cliente, Administrativo, Profesional.
Pagina Lista Usuarios.html 		Emite listado de Usuarios identificando su perfil.
Pagina Cliente.html 			Permite modificacion de datos usuario/perfil Cliente.
Pagina Administrativo.html 		Permite modificacion de datos usuario/perfil Administrativo.
Pagina Profesional.html 		Permite modificacion de datos usuario/perfil Profesional.
Pagina crearusuario.html		Permite crear un usuario en sistema.
Pagina editarcliente.html		Permite modificacion de datos usuario/perfil Cliente.
Pagina editaradministrativo.html	Permite modificacion de datos usuario/perfil Administrativo.
Pagina editarprofesional.html		Permite Editar.	
Pagina listadovisitas.html		Lista de todas las visitas realizadas a cada uno de los clientes.
Pagina responderchecklist.html		Listado de chequeos de una visita.
Pagina listadopago.html			Lista de todos los pagos realizados por los clientes.
Pagina crearpago.html			Formulario que permitirá agregar un pago al sistema.
Pagina listadoasesorias.html		lista con las asesorías realizadas.
Pagina crearasesorias.html		Formulario para agregar una nueva asesoría.
Pagina reportes.html			Sitio con reportes en forma de gráficos de diversos tipos.
Pagina administrarasistentes.html	Permite administrar asistentes/agregar. 
Pagina Login de usuario.html		Permite acceder a lista de usuarios. 
Pagina Listado de accidentes.html	Permite ingresar un evento de accidente a sistema.
